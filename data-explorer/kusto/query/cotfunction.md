# <a name="cot"></a>cot()

Berechnet den trigonometrischen Kotangens des angegebenen Winkels im Bogenmaße.

## <a name="syntax"></a>Syntax

`cot(`*Stuben*`)`

## <a name="arguments"></a>Argumente

* *x*: eine reelle Zahl.

## <a name="returns"></a>Rückgabe

* Der Kotangens-Funktionswert für `x`
