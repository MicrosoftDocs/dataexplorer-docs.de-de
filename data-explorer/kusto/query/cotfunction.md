# <a name="cot"></a>cot()

Berechnet die trigonometrische Kotangung des angegebenen Winkels in Bogenmaß.

**Syntax**

`cot(`*X*`)`

**Argumente**

* *x*: Eine reelle Zahl.

**Rückgabe**

* Der Kotangentenfunktionswert für`x`
