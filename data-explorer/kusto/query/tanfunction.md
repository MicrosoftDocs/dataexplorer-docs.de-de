# <a name="tan"></a>tan()

Gibt die Tangens Funktion zurück.

## <a name="syntax"></a>Syntax

`tan(`*Stuben*`)`

## <a name="arguments"></a>Argumente

* *x*: eine reelle Zahl.

## <a name="returns"></a>Rückgabe

* Das Ergebnis von `tan(x)`
