# <a name="sin"></a>sin()

Gibt die Sinusfunktion zurück.

**Syntax**

`sin(`*X*`)`

**Argumente**

* *x*: Eine reelle Zahl.

**Rückgabe**

* Das Ergebnis von `sin(x)`
