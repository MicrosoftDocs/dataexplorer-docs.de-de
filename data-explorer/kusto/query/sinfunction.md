# <a name="sin"></a>sin()

Gibt die Sinusfunktion zurück.

## <a name="syntax"></a>Syntax

`sin(`*Stuben*`)`

## <a name="arguments"></a>Argumente

* *x*: eine reelle Zahl.

## <a name="returns"></a>Rückgabe

* Das Ergebnis von `sin(x)`
