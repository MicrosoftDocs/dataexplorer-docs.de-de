# <a name="pow"></a>pow()

Gibt das Ergebnis der Erhöhung der Macht zurück

**Syntax**

`pow(`*base*`,` *Basisexponenten*`)`

**Argumente**

* *basis*: Basiswert.
* *Exponent*: Exponentenwert.

**Rückgabe**

Gibt die Basis zurück, die auf den Leistungsexponenten angehoben wird: Base - Exponent.
