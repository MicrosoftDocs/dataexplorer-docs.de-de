# <a name="new_guid"></a>new_guid()

Gibt eine zufällige GUID (Global Unique Identifier) zurück.

```kusto
new_guid()
```

## <a name="syntax"></a>Syntax

* `new_guid()` -generiert einen neuen Wert vom Typ `guid` .
