# <a name="cos"></a>cos()

Gibt die Cosinus-Funktion zurück.

## <a name="syntax"></a>Syntax

`cos(`*Stuben*`)`

## <a name="arguments"></a>Argumente

* *x*: eine reelle Zahl.

## <a name="returns"></a>Rückgabe

* Das Ergebnis von `cos(x)`
