# <a name="cos"></a>cos()

Gibt die Kosinusfunktion zurück.

**Syntax**

`cos(`*X*`)`

**Argumente**

* *x*: Eine reelle Zahl.

**Rückgabe**

* Das Ergebnis von `cos(x)`
